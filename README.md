# opencode-plugins
The repo for opencode artifacts bundles.

## yid-notes-assistant
Bundle for notes management. Contains 2 primary agents for adding notes and for reading them.
Bundle assumes using [telegram client for opencode](https://github.com/grinev/opencode-telegram-bot)
